# Software Testing Skill

This skill is based on the ISO/IEC/IEEE 29119 series of international software testing standards.

## Available Skill

### iso-29119
**Based on:** ISO/IEC/IEEE 29119 series (Parts 1-5)  
**Purpose:** Comprehensive software testing standards guidance  
**Use when:** User asks about testing concepts, processes, documentation, techniques, keyword-driven testing, or any software testing standard-related questions

## Standards Covered

The skill covers all five parts of the ISO/IEC/IEEE 29119 series:

- **Part 1:2013** - Concepts and definitions - Vocabulary and fundamental concepts
- **Part 2:2013** - Test processes - Multi-layer test process model
- **Part 3:2013** - Test documentation - Templates and guidelines
- **Part 4:2015** - Test techniques - Test design techniques
- **Part 5:2016** - Keyword-driven testing - Keyword-based test automation

## How to Use

The skill is located in: `.trae/skills/iso-29119/`

The skill contains a single `SKILL.md` file with comprehensive guidance based on all five ISO standards.
